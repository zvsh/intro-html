Introduction

HTML stands for HyperText Markup Language. It is the standard language for creating webpages. HTML describes the structure of webpages using markup.

Basic Syntax

HTML is made up of elements, which are enclosed in tags. Tags are enclosed in angle brackets, like this: <tag>. Most elements have an opening tag and a closing tag. For example:

<p>This is a paragraph.</p>

Elements

HTML elements can be nested inside one another. Here are some common elements:

  • <h1> to <h6> - Header tags, where <h1> is the largest and <h6> is the smallest.
  • <p> - Paragraph tag for blocks of text.
  • <a> - Anchor tag for links.
  • <img> - Image tag to display pictures.

Attributes

Attributes provide additional information about an element. They are written in the opening tag. For example:

<a href="https://www.example.com">Visit Example</a>

In the example above, href is an attribute of the <a> tag.

Examples

Creating a Simple Page

<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a simple webpage.</p>
</body>
</html>
